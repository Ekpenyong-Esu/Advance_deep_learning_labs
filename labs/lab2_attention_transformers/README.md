# Lab 2: Attention Mechanisms & Transformers

This lab implements the core components of the Transformer architecture from scratch using PyTorch.

## Topics

- Scaled Dot-Product Attention
- Multi-Head Attention
- Sinusoidal Positional Encoding
- Transformer Encoder Block
- Full Encoder stack for sequence classification

## Running the Lab

```bash
python main.py
```

The script trains a small Transformer encoder on a synthetic sequence-classification task and prints per-epoch accuracy.

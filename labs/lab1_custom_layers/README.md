# Lab 1: Custom Neural Network Layers

This lab demonstrates how to build custom neural network layers and architectures from scratch using PyTorch.

## Topics

- Depthwise Separable Convolutions
- Squeeze-and-Excitation (SE) Blocks
- Residual / Skip Connections
- A small ResNet-style classifier on CIFAR-10

## Running the Lab

```bash
python main.py
```

## Expected Output

Training progress will be printed to the console. After 10 epochs the model should achieve ~75% accuracy on the CIFAR-10 test set (CPU) and higher with GPU.
